ПРОФИЛИ ДЛЯ ГИПСОКАРТОНА

Профили для гипсокартона "RGP-Almaty" изготовлены из высококачественной оцинкованной стали с применением передовых технологий производства и успешно используются в строительстве «сухим способом», с применением гипсокартона (ГКЛ).

Правильная геометрия изделий RGP способствует слаженности и легкости работ при формировании каркасов перегородок, облицовок и подвесных потолков.

Стандартная длина профилей для гипсокартона "RGP" составляет 3 метра.

Монтаж подвесных потолков и каркасов стен из ГКЛ.

№ Наименование Фото схема Описание
1. ПП60x27 (SC, SE, ST, SX)

С-образный потолочный профиль ГКЛ
предназначен для формирования каркаса подвесных
потолков и облицовки стен. Полки и спинка профиля имеют
по три канавки для центровки ввинчиваемого шурупа и придают
ему дополнительную жесткость
2. ППН27х28 (SC, SE, ST, SX)

 

 Профиль направляющий потолочный U образный
служит в качестве направляющей ПП-профилей 60х27
при монтаже каркаса подвесного потолка, а также облицовки
стенгипсокартоном или иными отделочными материалами.
3. ПС50x50 (SE, ST, SX)

Стоечные профили имеют С-образную форму и служат
в качестве вертикальных стоек каркасов, предназначенных
для формирования гипсокартонных перегородок и облицовки
стенгипсокартонном. Монтируются в паре с соответствующими
по размеру направляющимипрофилями.
4. ПС75x50 (SE, ST, SX)

Стоечные профили имеют С-образную форму и служат
в качестве вертикальных стоек каркасов, предназначенных
для формирования гипсокартонных перегородок и облицовки
стенгипсокартоном. Монтируются в паре с соответствующими
по размеру направляющимипрофилями.
5. ПС100x50 (SE, ST, SX)

Стоечные профили имеют С-образную форму и служат
в качестве вертикальных стоек каркасов, предназначенных
для формирования гипсокартонных перегородок и облицовки
стенгипсокартоном. Монтируются в паре с соответствующими
по размеру направляющимипрофилями.
6. ПН50x40 (SE, ST, SX)

Профили имеют U-образную форму и служат в качестве
направляющих для стоечныхпрофилей, а также для устройства
перемычек между ними в каркасах перегородок и облицовок
изгипсокартона (ГКЛ). Монтируются в паре с соответствующими
по размеру ПС-профилем.
7. ПН75x40 (SE, ST, SX)

 

 Профили имеют U-образную форму и служат в качестве
направляющих для стоечныхпрофилей, а также для устройства
перемычек между ними в каркасах перегородок и облицовок
изгипсокартона (ГКЛ). Монтируются в паре с соответствующими
по размеру ПС-профилями.
8. ПН100x40 (SE, ST, SX)

Профили имеют U-образную форму и служат в качестве
направляющих для стоечныхпрофилей, а также для устройства
перемычек между ними в каркасах перегородок и облицовок
изгипсокартона (ГКЛ). Монтируются в паре с соответствующими
по размеру ПС-профилями.
9. Подвес прямой 60х27

Используется для крепления потолочных профилей к несущим
конструкциям. После крепленияпрофилей в проектное положение
выступающие концы прямого подвеса отгибаются или отрезаются.
10. Соединитель одноуровневый 60х27 (краб)

Соединитель потолочногопрофиля применяется для соединения
основного потолочного профиля (балки) с перпендикулярными
отрезками потолочного профиля для гипсокартона.
11. Удлинитель профилей 60x27

Используется для наращивания недостаточной длины профилей,
а также создает дополнительное ребро жесткости конструкции.
12. Подвес с зажимом 60x27

Применяется с тягой подвеса для крепления профилей ПП 60x27
к несущему основанию. При монтаже допускает регулировку
положения профилей по высоте до 4,5 м.
13. Соединитель двухуровневый 60x27

Служит для соединения между собой профилей ПП 60Х27
14. Тяга подвеса

Применяется для устройства анкерного (пружинного) подвеса.
При помощи подвеса с зажимом обеспечивает соединение
несущего потолка и каркаса из потолочного профиля для гипсокартона.
